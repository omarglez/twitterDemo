# appdirect-code-challenge-solution

Twitter feed demo

## Setup

Obtain starter code with your own yo generator.
Make sure the server runs.
Download this repo as a zip and copy all files into the code (replace all).

Install the project dependencies:

`npm install`

## Dev mode

In webpack.config.js, set watch to true.
This will allow you to change the code as the app is running and see the changes on refresh.

Then, run the app with:

`npm run dev`

## Running

Build the app:

`npm run build`

Starts the static and proxy servers:

`npm start`
